# summary

Initialize Tailwind CSS in a Salesforce project.

# description

Sets up Tailwind CSS for Lightning Web Components. Installs dependencies, creates configuration files, and adds runtime LWC components (TailwindElement base class and cn() utility).

# examples

- Initialize Tailwind CSS in the current project:

  <%= config.bin %> <%= command.id %>

# flags.output-dir.summary

Override the default package directory from sfdx-project.json.
